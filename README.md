# Rain-and-Gas-monitoring-System
This project presents an IoT-based solution for monitoring environmental conditions by detecting rainfall and harmful gases in real time. It uses sensors integrated with a NodeMCU (ESP8266) to collect data and transmit it to the Blynk cloud platform via Wi-Fi.

The system continuously monitors gas levels using an MQ2 sensor and detects rain using a rain sensor module. Whenever gas leakage or rainfall is detected, the system instantly sends alerts to the user through mobile notifications and email, enabling quick action even from remote locations.

This solution addresses the limitations of traditional systems by providing real-time remote monitoring, improving safety, and reducing risks in homes, industries, and agricultural environments. The Blynk mobile application offers a simple and user-friendly interface to visualize sensor data and receive alerts.

The system is designed to be cost-effective, energy-efficient, and easy to implement using affordable components. It can be further enhanced by integrating additional sensors, automation features, and predictive technologies for advanced environmental monitoring.
